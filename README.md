
THIS IS MY PROJECT TITLE - GROUND LEVEL WATER PREDICTION USING MACHINE LEARNING 

IN THIS PROJECT I - 
- Developed a machine learning–based system to predict groundwater levels using environmental, hydrological, and historical climatic data.
– Performed data preprocessing, feature engineering, and regression modeling to analyze groundwater trends and improve prediction accuracy.
– Evaluated model performance using standard metrics to ensure reliability and long-term behavior analysis.
– Built an interactive Streamlit web application for real-time visualization, analysis, and decision support in groundwater
resource planning.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
